## angular cli安装使用

##### 安装准备工作

1.安装Node，并确认装好npm

2.在PS（CMD）中，全局安装angular cli，默认自动选择最新版本安装

```shell
npm install -g @angular/cli
```

** 如果使用npm安装慢，则可以先安装cnpm，在使用cnpm执行安装

```shell
npm install -g @angular/cli
```

##### 测试安装

在PS（CMD）中，查看安装版本号

```powershell
ng --version
```
