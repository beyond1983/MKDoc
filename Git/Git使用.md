- [下载 git OSX 版](http://code.google.com/p/git-osx-installer/downloads/list?can=3)

- [下载 git Linux 版](http://book.git-scm.com/2_installing_git.html)

### Git配置



1.拉取远程仓库文件

> git pull origin master

origin是我为远程仓库取得名次，以便区分仓库

2.提交本地文件到缓存

添加所有文件

> git add *

添加制定文件

> git add 文件名称

添加提交说明

> git commit -m "提交说明"

3.提交文件到仓库

> git push
